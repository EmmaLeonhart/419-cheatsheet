Regression vs Classification
1) Regression problems:
▪we are trying to map input variables to some 
continuous function.
• Example: given data about the size of houses on 
the real estate market, try to predict their price. 
2) Classification problem:
▪we are trying to predict results in a discrete output. 
That is, we are trying to map input variables into discrete 
categories.
• Binary Classification: predict a binary label (e.g. cat vs. non-cat)
• Multiway classification: predict a discrete label (e.g. object category 
[for example, cat], from a list [for example, cat, dog, horse, bird])
▪For example, an SVM (Support Vector Machine) learns the 
boundary between data of two (or more) categories

Sample Code for Direct Solution
import numpy as np
from numpy.linalg import inv
# Input features
X= np.array([[1, 1], [1, 2], [1, 3], [1, 4]])
# Target values
Y = np.array([[2], [3], [4], [5]])
# Transpose of X
X_transpose = X.T
# Compute XT * X
X_transpose_X = np.dot(X_transpose, X)
# Compute XT * y
X_transpose_Y = np.dot(X_transpose, Y)
# Calculate the inverse of X_transpose_X
X_transpose_X_inv=inv(X_transpose_X);
# Solve the normal equation
w=np.dot(X_transpose_X_inv, X_transpose_Y)
# print the weights
print("w[0] = bias w₀:", w[0, 0])
print("w[1] = weight w₁:", w[1, 0])
# Input features for testing (1,1) and (1,4)
X_test = np.array([[1,1], [1,4]])
predictions = np.dot(X_test, w)
print(predictions)

def model(X, w):
 return w[0] + X * w[1] #w is shape (2,1): w[0]=bias, w[1]=weight
#caluclate the activation (probability)
def sigmoid(t):
  return 1.0 / (1.0 + np.exp(-t))
#caluclate the loss
def cross_entropy(w, X, Y, eps=1e-12):
  #predicted probabilities (Y_hat)
  a = sigmoid(model(X, w))
  #cross entropy: -( Y log(a) + (1-Y) log(1-a) )
  loss = -np.mean(Y * np.log(a) + (1 - Y) * np.log(1 - a))
  return loss

#Find the gradient based on current weights – single step at k iteration
def grad_cross_entropy(w, X, Y):
  P = X.shape[0]
  a = sigmoid(model(X, w))       #P×1
  error = a - Y            #P×1
  dw0 = np.sum(error) / P       #scalar
  dw1 = np.sum(error * X) / P     #scalar
  grad = np.array([[dw0], [dw1]])   #2×1
  return grad
#The gradient descent algorithm – full iterative
def gradient_descent(X, Y, alpha=0.5, iters=2000):
  w = np.zeros((2,1))         #random initialization , start at [0,0]
  loss_history = []
  for k in range(iters):
    loss = cross_entropy(w, X, Y)
    loss_history.append(loss)
        #find the gradient
    g = grad_cross_entropy(w, X, Y)
    w = w - alpha * g
    if k % 200 == 0:
      print(f"iter {k:4d} | loss = {loss:.6f} | w0={w[0,0]:.4f}, w1={w[1,0]:.4f}")
  return w, np.array(loss_history)
############################# CELL 5
